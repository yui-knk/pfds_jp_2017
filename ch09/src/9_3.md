# ねじれ二進数のメモ

各桁の重み:

```
i = 0 :  1
i = 1 :  3
i = 2 :  7
i = 3 : 15
i = 4 : 31
i = 5 : 63
```

`002Y...`をインクリメント(+1)するときは、`2(2^(2+1) - 1) + 1`なので、

```
  2(2^(2+1) - 1) + 1
= 2^(3+1) - 1
```

であり、`000(Y+1)...`となる。

0から15までの表現は以下のとおり(左から、10進数、密な表現、ねじれ二進数):

```
 0 : []        : 0
 1 : [1]       : 1
 2 : [1,1]     : 2
 3 : [3]       : 01
 4 : [1,3]     : 11
 5 : [1,1,3]   : 21
 6 : [3,3]     : 02
 7 : [7]       : 001
 8 : [1,7]     : 101
 9 : [1,1,7]   : 201
10 : [3,7]     : 011
11 : [1,3,7]   : 111
12 : [1,1,3,7] : 211
13 : [3,3,7]   : 021
14 : [7,7]     : 002
15 : [15]      : 0001
```

## デクリメントについて

例えば、`14 -> 13`。
最も下の桁から1を引いて、2つに分解すればいいので、`7 -> (3,3)`。
