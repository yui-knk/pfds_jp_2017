## P.99

`[1,2,3]`の例:

```
WORKING ([1,2,3], [])
WORKING ([2,3], [1])
WORKING ([3], [2,1])
WORKING ([], [3,2,1])
DONE ([3,2,1])
```

## P.100 (AppendState)

`append [1,2], [3,4]`の例:

```
REVERSING ([1,2], [], [3,4])
REVERSING ([2], [1], [3,4])
REVERSING ([], [2,1], [3,4])
APPENDING ([2,1], [3,4])
APPENDING ([1], [2,3,4])
APPENDING ([], [1,2,3,4])
DONE ([1,2,3,4])
```

## P.100 (RotationState)

`rotate [1,2], [5,4,3]`の例:

```
REVERSING ([1,2], [], [5,4,3], [])
REVERSING ([2], [1], [4,3], [5])
REVERSING ([], [2,1], [3], [4,5])
APPENDING ([2,1], [3,4,5])
APPENDING ([1], [2,3,4,5])
APPENDING ([], [1,2,3,4,5])
DONE ([1,2,3,4,5])
```
